# Rosewood College — Logo & Branding Proposal

Interactive web presentation for client review.

## Structure

- **Cover** — Title slide
- **Option 1 (The Living Tree)** — Divider → Logo + rationale → Applications (uniforms, signage, brand)
- **Option 2 (The Heritage Rose)** — Divider → Logo + rationale → Applications

